Modeling cell type colocalization in multiplex imaging data using functional regression and wild bootstrap

Carly Middleton, Qi Zheng, Jun Yan, Susmita Datta, Maiying Kong

07 April 2026

This vignette introduces the R package funboot. In the following, we demonstrate how to 1.) use in-package functions to preprocess single-cell spatial data, 2.) compute spatial summary functions and colocalization curves for pairs of cell types, and 3.) quantify colocalization as a function of patient-level and spatial covariates, using the concurrent functional regression framework and our wild bootstrap-based inference procedure.

We illustrate this workflow by reproducing our manuscript’s real data analyses of the publicly available Jackson-Fischer breast cancer and Hoch-Schulz melanoma imaging mass cytometry datasets.

Although a minimum of B1 = 50 outer and B2 = 50 inner bootstrap samples are generally recommended in practice, this vignette uses B1 = B2 = 10 and n_perm= to reduce computation time. As a result, the vignette takes about ___ minutes to build on a standard laptop computer with 8 cores, a 2.3 GHz processor, and 16 GB of RAM.

library(patchwork)

Breast Cancer Data

The 100-image subset of the Jackson-Fischer data contains 1 image per subject. It is stored in-package and can be loaded as follows:

data("breastcancer_data", package='funboot')

We focus on cell metaclusters 3 (T cells), 7 (endothelial cells), 9 (small circular stromal cells), and 16 (proliferative epithelial cells). Their spatial distributions can be examined using funboot::plot_images():

Cell types 7 (endothelial cells) and 3 (T cells) appear to be colocalized. We can examine their colocalization curve by fitting the model

\[ Y_{i}(r) = \beta_0(r) + \epsilon_i(r), \qquad i = 1, \ldots, n, \] where \(Y_{i}(r)\) denotes the observed colocalization curve for T cells around endothelial cells within the image belonging to subject \(i\), and performing inference on the intercept \(\beta_0(r)\). The first step is to discard images with sparse cell counts and then calculate the observed and expected spatial summary function. Because some of these images contain holes, we apply the permutation adjustment during the calculation of the expected spatial summary function. The process can be carried out using funboot::preprocess_data().

set.seed(12345)
sumfun_data_interaction <- funboot::preprocess_data(data=breastcancer_data, 
                                                    from_cell=7, 
                                                    to_cell=3,
                                                    qc_cellcount_cutoff=20, 
                                                    n_perm=5, 
                                                    perm_yn=TRUE,
                                                    r_max=500, 
                                                    inc=1, 
                                                    image_dims=c(0,1000,0,1000),
                                                    summary_function='L')

we can calculate and plot a confidence band for the sum of both intercepts using funboot::wildBS_CB() and funboot::plot_wildBS_CB().

CB_interaction <- funboot::wildBS_CB(formula=outcome ~ 1,

Because the confidence band crosses zero, the conclusion would be that a significant amount of interaction is present. (However, many more bootstrap samples are needed for accurate results.) The above can also be repeated for the avoidance example.

Next, we incorporate covariates into the model:

sumfun_data_covariates <- preprocess_data(data=breastcancer_data,
                                           from_cell=7,
                                           to_cell=3,
                                           qc_cellcount_cutoff=20,
                                           n_perm=5,
                                           perm_yn=T,
                                           r_max=500,
                                           inc=1,
                                           image_dims=c(0,1000,0,1000),
                                           summary_function='L',
                                           verbose=TRUE)
#combine cell types 3,4,5,6 into 1 category
#breastcancer_data$cell_type <- ifelse(breastcancer_data$cell_type %in% c(3:6), 3,
#                                                       breastcancer_data$cell_type)
spatialcov_data <- preprocess_data(data=breastcancer_data,
                                   from_cell=7,
                                   to_cell=7,
                                   qc_cellcount_cutoff=20,
                                   n_perm=5,
                                   perm_yn=T,
                                   r_max=500,
                                   inc=1,
                                   image_dims=c(0,1000,0,1000),
                                   summary_function='L',
                                   verbose=TRUE)
names(spatialcov_data)[names(spatialcov_data)=='L_obs'] <- 'spatial_cov'
spatialcov_data$L_expect <- spatialcov_data$L_pmean <- spatialcov_data$outcome <- NULL
sumfun_data_full_covariates <- merge(sumfun_data_covariates,
                                     spatialcov_data,
                                     by=c('image_number','r','patient_id','patient_age','tumor_grade'),
                                     all=T)
df <- data.frame(image_number = rep(unique(breastcancer_data$image_number), each=501),
                 r = rep(0:500, length(unique(breastcancer_data$image_number)))
                 )
sumfun_data_full_covariates <- merge(df,
                                     sumfun_data_full_covariates,
                                     by=c('image_number','r'),
                                     all=T)
sumfun_data_full_covariates <- sumfun_data_full_covariates[!is.na(sumfun_data_full_covariates$outcome),] #Drop obs. with missing outcomes
sumfun_data_full_covariates$spatial_cov <- ifelse(is.na(sumfun_data_full_covariates$spatial_cov), 0,
                                                  sumfun_data_full_covariates$spatial_cov) #set L^{3,16}(r) = 0 when there are no type 16 cells
sumfun_data_full_covariates$grade2 <- ifelse(sumfun_data_full_covariates$tumor_grade == '2', 1,
                                             ifelse(is.na(sumfun_data_full_covariates$tumor_grade), NA,
                                                    0))
sumfun_data_full_covariates$grade3 <- ifelse(sumfun_data_full_covariates$tumor_grade == '3', 1,
                                             ifelse(is.na(sumfun_data_full_covariates$tumor_grade), NA,
                                                    0))

Calculating the p-value for the test of the spatial covariate:

CB_beta4 <- wildBS_CB(formula=outcome ~ patient_age + grade2 + grade3 + spatial_cov,
                      data=sumfun_data_full_covariates,
                      spatial_covars = c("spatial_cov"),
                      target = c("spatial_cov"),
                      form   = c("coef"),
                      covar_list = list(),
                      func = function(spatial_cov){
                        spatial_cov
                      },
                      B1=10,
                      B2=10,
                      alpha=.05,
                      re=NULL,
                      nthreads = 1,
                      id='patient_id')
M <- max(abs((CB_beta4$target_curve_estimates - 0)/CB_beta4$target_curve_SEs))
mean(CB_beta4$M>M) #p-value for the test of beta4(r)=0
#> [1] 0
